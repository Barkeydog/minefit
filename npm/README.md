# minefit npm packages

This folder is used by release automation to build the platform-specific npm binary packages:

- `@barkey/minefit-win32-x64-msvc`
- `@barkey/minefit-darwin-arm64`
- `@barkey/minefit-darwin-x64`
- `@barkey/minefit-linux-x64-gnu`

Package contents are generated during release publishing and are not meant to be edited manually.
